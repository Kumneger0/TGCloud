# Changelog – 2026-02-22

## 🚀 New Features

- **Global Media Player**: Seamlessly listen to audio across the entire app with a new global player and loading states.
- **Enhanced Video Streaming**: Smooth playback for even the largest video files using advanced streaming technology (MSE).
- **Flexible Connections**: Choose between connecting your personal Telegram account to bypass bot limits or using a standard bot connection.
- **Improved File Tracking**: New alerts help you find files that might have been moved or removed in Telegram, with direct search links.
- **File Sync**: You can now sync files you've uploaded directly to Telegram outside of TGCloud — everything stays in one place.

## 🌟 Improvements

- **Refined User Interface**: Optimized global modals and dark mode visuals for a more premium look and feel.
- **Better Performance**: Faster app loading and more reliable Telegram connections thanks to technical optimizations.
- **Detailed Media Info**: Video durations and other media details are now more consistently displayed.

## 🐞 Bug Fixes

- Fixed issues where some video durations were not showing up.
- Resolved a bug that prevented deleting files from Telegram in some cases.
- Fixed a bug where audio would keep playing after closing the file viewer.
- Fixed an issue where switching between audio tracks wasn't working correctly.
